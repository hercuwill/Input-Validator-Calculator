# Input-Validator-Calculator
A Java application demonstrating comprehensive input validation and conditional arithmetic operations across multiple data types.
## Features
- Validate String input (non-empty names)
- Validate integer input (range: 10-30)
- Validate floating-point input (range: 0.44-0.66)
- Perform different operations based on input values
- Personalized output with user's name
- Easter egg for specific names

## How to Run
1. Compile: `javac Project1.java`
2. Run: `java Project1`
3. Follow the prompts to enter your name, an integer, and a decimal number

## Skills Demonstrated
- Input validation across multiple data types
- Error handling with informative messages
- Conditional logic with if-else statements
- Arithmetic operations (addition, subtraction, division, modulo, multiplication)
- String comparison methods
